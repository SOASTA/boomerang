BOOMR.utils. MD5

Adds support for MD5 to Boomerang (as BOOMR.utils.md5).

For information on how to include this plugin, see the Building tutorial.

Beacon Parameters

This plugin adds no parameters to the beacon.

Methods

md5(string [, key] [, raw])

Calculates the MD5 of the specified string and optional key.

Parameters:

Name Type Argument Description
string string

Input string
key string <optional>

Key
raw boolean <optional>

Whether or not to return data raw MD5 (versus hex-encoded)

Returns:

Type: string

Raw or hex-encoded MD5 of the input string and key
